# Troubleshooting

## Budget Not Evaluating

Wait several hours.

Budget evaluations are not immediate.

---

## Email Not Received

Check:

- Spam Folder

- Logic App Run History

- Outlook Connection

---

## Logic App Not Triggering

Verify:

HTTP Trigger

↓

Publish Workflow

↓

Action Group Configuration

---

## Budget Shows Action Group: None

Azure Cost Management Budgets under Billing Account scope may not display Action Groups.

This is expected behavior.

---

## Permission Issues

Ensure you have Contributor or Owner access to the Azure Subscription.
