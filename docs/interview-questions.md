# Interview Questions

## 1. What is Azure Cost Management?

Azure Cost Management is a service used to monitor, analyze and optimize Azure cloud spending.

---

## 2. What is an Azure Budget?

Azure Budget allows organizations to define spending limits and receive alerts when thresholds are exceeded.

---

## 3. What is Azure Monitor?

Azure Monitor collects and analyzes telemetry from Azure resources.

---

## 4. What is an Action Group?

An Action Group defines the notification or automation executed when an alert is triggered.

---

## 5. Why use Logic Apps?

Logic Apps provide a no-code/low-code workflow automation platform for integrating Azure services.

---

## 6. What is FinOps?

FinOps is the practice of managing and optimizing cloud costs through financial accountability and operational efficiency.
